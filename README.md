# aoc-2022

My solutions to [Advent of Code 2022](https://adventofcode.com).

### Setup

Clone
```
git clone https://github.com/davidwrenner/aoc-2022.git
```

Run daily solution
```
cd src
python XX.py  # requires ../input/XX.txt
```
